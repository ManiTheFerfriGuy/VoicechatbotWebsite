# VoiceChatBot Shop Landing Page

یک صفحه فرود فارسی و مدرن برای فروشگاه ربات دستیار صوتی VoiceChatBot.

## اجرا

1. مخزن را کلون کنید و وارد پوشه شوید.
2. یک سرور ساده محلی اجرا کنید (مثلاً با Python):

```bash
python -m http.server 8000
```

3. در مرورگر به آدرس `http://localhost:8000` بروید و فایل `index.html` را مشاهده کنید.

## ساختار فایل‌ها

- `index.html`: ساختار صفحه و محتوای فارسی.
- `styles.css`: استایل مدرن، تاریک و مینیمال با تایپوگرافی فارسی.
- `script.js`: اسکرول نرم، هایلایت منو و درج خودکار سال.
